      1. Use your rootfinder to write a function that does the following: Given the temperture and mixing ratio at height z=0 find the temperture as a function of height assuming saturated adiabatic ascent.

        For this exercise:

        1. Start at z0=0, p0 = \(10^5\) Pa, T0=280 K

        2. Calculate an initial \(h_{m0}=c_p T_0 + l_v r_{sat}(T_0, p_0 ) + g z_0\) using \(l_v = 2.5 \times 10^6\ J/kg/K\) and \(c_p = 1004\ J/kg/K\) and g=9.8 \(m\,s^{-2}\) (eqn 38 of the `thermodynamics notes`_ where I unfortunately called the moist static energy \(s_v\) instead of \(h_m\) ) Use the definition on p. 5 of the `thermodynamics notes`_ that \(r_{sat}(T, p ) = \rho_{sat}/\rho_{d} = \epsilon e_{sat}/(p - e_{sat} )\) where \(\epsilon = R_d / R_v = 287 (J/kg/K) / 461 (J/kg/K) = 0.622\)

        3. Use the rootfinder to find a vector of temperatures that satisfies \(h_m = h_{m0}\) for a hydrostatic atmosphere where the pressure decreases according to \(dp = - \rho g dz\) given a vector of heights from z=0 to z=3000 m in 10 m increments, with \(\rho = p/(R_d T)\) and \(R_d\) = 287 J/kg/K

      2. Use this function to calculate T,z,rv,and p for a saturated hydrostatic atmosphere and show that \(\theta_e\) is approximately constant.
